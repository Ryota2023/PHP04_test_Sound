# PHP4回目 SESSION機能
●提出日：2023/7/13

##　画面を開いたら自動的に音を鳴らす方法がわかりませんでした。

今回の課題PHP4回目では、ログイン機能を追加しましたが、
ログインID,PWを失敗したときに音（.sound/beep_04.wav）を鳴らすようにしたかったのですが、
残念ながらできませんでした。

PHP上で音が鳴るのはボタンを押した後に鳴ったので確認できたのですが、
画面が開いてすぐ鳴らすことができなかったです。

ログイン失敗したときに、赤文字で「ログイン失敗しました」と表示されたときに
ピー音を鳴らしたいです。

##　ファイル名
・ログイン画面はトップ画面のindex.php
・音源は/sound/beep_04.wav
・音が鳴ることを確認したファイル名→ /sound3.php
